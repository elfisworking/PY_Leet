# Leetcode上关于二分搜索的题目
注意答案的二段性：
如果可以确定答案在一个[left,right]区间内，那么就要考虑使用二分搜索来解决答案
模板:
```python
class Solution:
    def problem():
        left , right = x, xx # 确定left ,right
        # 判断limit 是否是解
        def check(limit):
            ....
        if 满足某种条件:
            return True
        return False
        
        # 二分搜索
        while left < right:
            mid = (left+right)//2
            if check(mid 或者其它跟mid相关的):
                right = mid
            else:
                left = mid + 1
        return left


```
